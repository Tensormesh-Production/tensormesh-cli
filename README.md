# Tensormesh CLI

Public standalone distribution repo for the `tm` CLI.

Install latest:

```bash
curl -fsSL https://raw.githubusercontent.com/Tensormesh-Production/tensormesh-cli/main/install-tm.sh | sh
```

Pin a specific release:

```bash
curl -fsSL https://raw.githubusercontent.com/Tensormesh-Production/tensormesh-cli/main/install-tm.sh | TM_VERSION=vX.Y.Z sh
```

The source for this distribution is released from the private `Tensormesh-Production/tensormesh-sdk` repository.
